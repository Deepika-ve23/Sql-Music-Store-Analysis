# Sql-Music-Store-Analysis
Project In Sql
🎵 SQL Music Store Data Analysis
📌 Project Overview

This project focuses on analyzing a Music Store Database using SQL to extract meaningful business insights.
It is designed to demonstrate strong data analysis, SQL querying, and problem-solving skills—perfect for aspiring Data Analysts.

The goal is to help the business understand customer behavior, sales trends, and revenue generation.

🛠️ Tools & Technologies
PostgreSQL
PgAdmin 4
SQL (Structured Query Language)
🗂️ Database Schema

The database consists of multiple tables such as:

Customers
Invoices
Invoice Items
Tracks
Albums
Artists
Genres

These tables are interconnected to simulate a real-world music store.

📊 Key Analysis Performed

This project answers important business questions like:

Who are the top customers by spending?
Which countries generate the most revenue?
What are the most popular music genres?
Which artists have the highest sales?
What is the best-selling track?
🧠 SQL Concepts Used
SELECT, WHERE, ORDER BY
GROUP BY, HAVING
JOINs (INNER JOIN, LEFT JOIN)
Aggregate Functions (SUM, COUNT, AVG)
Subqueries
Window Functions
